# Personality Psychology: Advanced Analysis & Insightful Visualizations

A light-hearted project, in Python using Pandas and NumPy, by a Data Analyst and aspiring Data Scientist utilizing analytical & modeling skills to dig deep into a dataset on personality psychology and people's traits and psychometrics.

Visuals were created in the form of bar graphs and scatterplots, through the beautiful and color-adding Seaborn library in addition to Matplotlib.
